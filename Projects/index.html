My Projects

Spatial Analysis of Tweets

I developed a Python script that ran on a RaspberryPi for several weeks collecting geo-referenced tweets that had terms relating to "ISIS". Results were stored a MySQL database. The tweets were then extracted from the database and analyzed using ArcGIS and Excel to find spatial and temporal patterns and statistics.

Appalachian Trail Mapbook

The land-use-land-cover (LULC) for the Appalachian Trail (AT) was analyzed to see how much of the different vegetation types exist along the trail. LULC data from 1992 was compared to LULC data from 2006 to see if there were any significant changes. A map book was also developed which shows sections of the trail and key attributes along the way.

Live Webmap of Tweets

I developed a Python script using the Flask micoframework to fetch geo-referenced tweets based on a specific tag. The tweets are pushed to leaflet webmap in near real-time via websockets. You can find the code here. Note: the Twitter API has since changed and this may not work.

ScriptEx

ScriptEx is designed to execute python files and log the output to a file. It accepts single python files, zip files, or directory paths as commandline parameters. It uses JSON config files to store a series of configurations to run the files with. This includes interactive inputs as well as commandline arguments. It's provided as is. Only tested on Windows. I wrote this to run a batch of student homework assignments at once rather than having to manually test each one.

Locating Low-Altitude Obstacles for Drone Navigation using Lidar

My main thesis research involves locating Medical Drones to reduce EMS response times. I was interested in determining the impact that obstacles at low altitudes (buildings & trees) have on drone flight times and range. Morphological image processing was used in conjunction with ArcGIS to extract buildings and trees from Lidar data.

Windroses with D3.js

The D3.js library was used to create an interactive windrose and an pollutionrose.

Raspberry Pi Controlled Christmas Lights

I developed a python application using Flask and the RaspberryPi to control a box of 8 outlets which are connected to strands of Christmas lights. The lights can be "synced" to music using lightshowPi The lights can also be turned on or off using a simple secured web interface.

Kinect Recording & Gesture Recognition

As part of my teams senior design project, I implemented a simple C++ application to record skeletons. KinectRecording is a package of programs which can be used to record and recognize hand/upperbody gestures from a Kinect. It relies on the OpenNI SDK to interface with the original Microsoft Kinect.

EEG Volume Control

This was a team project to use EEG (brain signals) to control the volume of a computer. The BioRadio was used in conjuction with MATLAB to record EEG signals while a person listened to loud and quiet music. Support Vector Machines and Radial Basis Function Networks were used to classify the EEG signals. Unfortunatly, we were unable to get real-time classification working. The Github Repo is available here.

My Research

Out-of-hospital cardiac arrest (OOHCA) is prevalent in the United States. Each year several hundred-thousand people die due to cardiac arrest. The automated external defibrillator (AED) has greatly enhanced survival rates for OOHCA. However, one of the most important factors in successful resuscitation is emergency medical services (EMS) response time. Unmanned aerial vehicles have routinely been used for remote sensing but there are new opportunities to use drones for medical emergencies. The objective of my research is examine spatial variation in cardiac arrest incidence rates in Salt Lake County and to provide a framework for locating a network of AED enabled drones to significantly reduce response times and therefore increase survival rates for OOHCAs. I've worked with my advisor to develop a new location model to determine the most cost-effective way to cover 90% of the demand for cardiac arrest medical service within one minute and five minute standards. The demand for service was derived from empirical OOHCA incident data.

I heavily use GIS and Python to determine the demand and potential facilities. Once the demand and potential areas are determined, we can apply models such as MCLP and LSCP to determine the optimal configuration of drones. The models are then solved using Gurobi, a mathmatical programming solver. Results can then be mapped and analyzed.
